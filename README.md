# interactive-dictionary
Interaktivní slovník | HTML, CSS, PHP, MySQL Webová aplikace pro správu a procvičování slovní zásoby. Uživatel může přidávat, upravovat a mazat slova uložená v databázi MySQL. Backend zpracovává formuláře v PHP, frontend je postaven na HTML/CSS s dynamickými prvky. Projekt zahrnuje návrh databázového schématu a propojení se serverovou logikou.
